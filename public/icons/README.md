# PWA Icons

This directory should contain PWA icons in the following sizes:

- `icon-16x16.png`
- `icon-32x32.png`
- `icon-72x72.png`
- `icon-96x96.png`
- `icon-128x128.png`
- `icon-144x144.png`
- `icon-152x152.png`
- `icon-192x192.png`
- `icon-384x384.png`
- `icon-512x512.png`

## Generating Icons

You can generate these icons from a source image using tools like:

1. **PWA Asset Generator**: https://github.com/nicnocquee/pwa-asset-generator
2. **Real Favicon Generator**: https://realfavicongenerator.net/
3. **PWA Builder**: https://www.pwabuilder.com/

## Icon Requirements

- Icons should be square PNG images
- Use a transparent background or solid color that matches your theme
- Ensure the icon is visible on both light and dark backgrounds
- For maskable icons, keep important content within the safe zone (center 80%)
